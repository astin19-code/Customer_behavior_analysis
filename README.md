Customer Shopping Behavior Analysis

A complete Customer Shopping Behavior Analysis Project developed using Power BI and Python to explore customer purchasing patterns, sales trends, seasonal performance, and business insights through interactive visualizations and data analysis.

📌 Project Overview

This project focuses on analyzing customer shopping behavior to understand how different factors such as demographics, product categories, seasons, subscriptions, and discounts influence purchasing decisions.

The project combines:

📊 Power BI Dashboards for interactive business intelligence
🐍 Python Analysis for data processing and visualization
📈 Data-driven insights for improving customer engagement and sales strategies
🎯 Objectives
Analyze customer purchasing behavior
Identify top-performing product categories
Study seasonal sales trends
Understand customer demographics
Generate meaningful business insights
Build interactive dashboards for decision-making
